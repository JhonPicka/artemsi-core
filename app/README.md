# ARTEMSI — application Next.js

Code source : `src/` (App Router, API routes, composants, `lib/`).

## Demarrage

Depuis ce dossier :

```bash
npm install
cp .env.example .env.local   # puis remplir les variables
npm run dev                  # http://localhost:3000
```
